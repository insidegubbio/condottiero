# { title }

Glavni meni, koji se nalazi na vrhu interfejsa, pruža pristup radnjama, opcijama i podešavanjima podeljenim u nekoliko kategorija, objašnjenih odvojeno u sledećim odeljcima.

<DocsNote>

Većina radnji u meniju može se izvršiti i pomoću prečica na tastaturi prikazanih u meniju.

</DocsNote>
