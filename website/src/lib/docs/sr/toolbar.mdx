# { title }

Traka sa alatkama se nalazi na levoj strani mape i predstavlja srce aplikacije, jer omogućava pristup glavnim funkcijama **gpx.studio**.
Svaki alat je predstavljen ikonom i može se aktivirati klikom na nju.

<div class="flex flex-row justify-center text-foreground">
    <div>
        <Toolbar class="border rounded-md shadow-lg" />
    </div>
</div>

Kao i kod [akcija uređivanja](./menu/edit), većina alata se može primeniti na više datoteka odjednom i na [unutrašnje tragove i segmente](./gpx).

Sledeći odeljci detaljno opisuju svaki alat.
